# Mission: AI Agent 开发与前沿研究

## Why
我正在从「会调 LLM API」走向「能独立设计、构建并评估 AI Agent」。目标有两条线：
一是**求职/转行**——进入 AI 应用开发 / AI 工程师相关岗位，需要能上手做出东西、并在面试中讲清原理；
二是**前沿研究**——具备读懂 Agent 相关论文、判断某项新技术「是真突破还是营销话术」的能力，不被带节奏。
学会之后，我能独立交付一个可展示的 Agent 项目，并对这个快速变化的领域建立自己的判断力。

## Success looks like
- 能用 Python **从零手写**一个带「工具调用 + 记忆 + 循环」的 Agent，且不依赖重型框架也能讲清每一行为什么这样写。
- 能清晰讲出 **ReAct、tool use、planning、memory、evaluation** 等核心机制，并能在面试中表达。
- 能**读懂一篇** Agent 领域的关键论文（如 ReAct），复述其核心贡献与局限。
- 拥有一个**可展示的个人 Agent 项目**（portfolio），GitHub 可访问。
- 建立一份**前沿跟踪习惯**：知道去哪里看新论文、如何快速判断是否值得深入。

## Constraints
- 每周可投入 **7-15 小时**。
- 技术栈锁定 **Python** 生态。
- **已有** LLM API 调用 / prompt 编写基础，不想在编程基础或 prompt 入门上重复。
- 偏好**理论 + 动手**结合；喜欢被「恰到好处地挑战」（desirable difficulty）。
- 授课语言：**中文**。

## Out of scope（暂时不碰，保护专注度）
- 从零训练 / 微调大模型本身（那是另一个方向，先建立 Agent 工程能力）。
- 硬件部署、大规模推理工程细节。
- 非 Python 生态（LangChain.js 等）作为主线，仅在对比时提及。

---
