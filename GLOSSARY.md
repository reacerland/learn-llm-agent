# AI Agent 术语表 (Glossary)

本工作区的**官方语言**。所有课程、练习、学习记录都遵循这里的术语。
> 规则：只有当用户**真正理解**一个术语后，才把它收录进来。这是一个「已压缩的知识」记录，不是用来从头读的字典。它会随学习推进持续生长。

## 基础（用户已具备，作为词汇锚点）

**LLM (大语言模型)**:
基于海量文本训练、能根据上文预测下一个 token 的概率模型；是 Agent 的「大脑」。
_别名/避免_：避免与「聊天机器人」混为一谈——聊天只是它的一种用法。

**Prompt (提示词)**:
给 LLM 的输入指令与上下文。一次 LLM 调用 = prompt in → 文本 out。

**Token**:
LLM 处理文本的最小单位（约等于一个字或半个词）。计费、上下文长度都以 token 计。

**Context window (上下文窗口)**:
模型一次能「看见」的 token 上限。超出则需截断或借助外部记忆。

**API call (一次模型调用)**:
向模型发送 prompt、接收输出的无状态请求。**这是「普通 LLM 调用」的基本单元**，也是 Agent 循环里的一步。
